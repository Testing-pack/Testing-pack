Ключевые пояснения к схеме:

Пропорция (бинарная): Z-тест для пропорций — стандартный выбор. Можно использовать хи-квадрат. Для малых выборок — точный тест Фишера.

Непрерывное среднее:

Параметрический путь (t-тест): Самый мощный, но требует проверки условий (нормальность остатков, гомоскедастичность). На практике часто устойчив к умеренным отклонениям от нормальности, особенно при больших выборках (N > 30-50 в каждой группе, благодаря ЦПТ).

Непараметрический путь (Манна-Уитни): Используется при сильных нарушениях нормальности, наличии выбросов или для ранговых данных. Важно: он проверяет сдвиг в «расположении» распределений, а не разницу средних.

Отношение (A/B): Самый сложный случай (например, средний чек = сумма / количество, CTR = клики / показы). Прямое применение t-теста к средним опасно.

Линеаризация (дельта-метод): Преобразование метрики в линейную форму, к остаткам которой уже можно применять t-тест. Ваш скриншот с yᵢ = B * xᵢ + ε — именно об этом.

Bootstrap: Универсальный, но computationally intensive метод для построения доверительных интервалов.

Poisson-bootstrap: Особенно хорош для редких событий (покупки, клики).

Ранговая / Нестандартная: Медиана, перцентили, нечисловые данные. Классический выбор — Манна-Уитни. Для сравнения более двух групп или с учетом стратификации — Permutation tests или расширения.

H₀: Конверсия в контрольной группе = конверсии в тестовой группе. (CR_cont = CR_test)

H₁: Конверсия в группах различается. (CR_cont ≠ CR_test) — двусторонняя, или CR_test > CR_cont — односторонняя.

Уровень значимости (α): Обычно 0.05 (5% риск false positive для одной целевой метрики).

Мощность (1-β): Обычно 0.8 или 0.9 (80-90% шанс обнаружить эффект, если он есть).

Подбор размера выборки:
Ваши формулы верны для базовых случаев.

Для средних (t-тест): n = 16 * σ² / Δ² (где σ — оценка стандартного отклонения, Δ — минимальный детектируемый эффект, MDE). Формула для мощности 0.8 и α=0.05.

Для пропорций (z-тест): n = (p1*(1-p1) + p2*(1-p2)) / (p1-p2)² * (Z_{1-α/2} + Z_{1-β})²

Важно: В реальности все сложнее (CUPED, стратификация, прогнозирование дисперсии). Необходимо учитывать:

Трафик и длительность эксперимента.

Стратификацию (sizing): Включать в формулы поправку на внутристратовую корреляцию. Это уменьшает требуемый размер выборки.

Для отношений (средний чек) расчет сложнее, часто делается через линеаризацию.

Поправки на множественность (Множественное тестирование):
Критически важны для заградительных метрик!

Поправка Бонферрони: α_new = α / m (где m — количество гипотез). Слишком консервативна, сильно снижает мощность.

Поправка Холма (Holm-Bonferroni): Менее консервативна. Ранжируем p-values по возрастанию: p₁ ≤ p₂ ≤ ... ≤ p_m. Сравниваем pᵢ с α / (m - i + 1). Первую неотвергнутую гипотезу и все следующие — сохраняем.

Для целевой метрики поправка обычно не нужна (если она одна).

Доверительные интервалы:
Для пропорций: p̂ ± Z * sqrt( p̂*(1-p̂) / n )

Для средних (t-тест): x̄ ± t_{α/2, df} * (s / sqrt(n))

Относительное изменение (lift): (p_test / p_control - 1) * 100% + доверительный интервал через дельта-метод или Fieller's theorem.

Ваш скриншот с β ± Z * SE — это доверительный интервал для линеаризованной модели после регрессии.

Примите решение:

Целевая метрика: Стат. значимый положительный эффект (с учетом направления) → победа.

Заградительные: Ни одна не должна показать значимого ухудшения после поправок.

Если целевая выигрышна, а заградительные не проигрышны → можно запускать.
